# 키보드 단축키

## 전체 단축키 목록

| 단축키 | 기능 | 범주 |
|--------|------|------|
| Ctrl+R | 파이프라인 시작 (Run) | 파이프라인 |
| Ctrl+S | 파이프라인 정지 (Stop) | 파이프라인 |
| Ctrl+Q | 애플리케이션 종료 | 파일 |
| Alt+1 | Status Dashboard 탭 | 탭 전환 |
| Alt+2 | Frame Preview 탭 | 탭 전환 |
| Alt+3 | Configuration 탭 | 탭 전환 |
| Alt+4 | Simulator Control 탭 | 탭 전환 |
| Alt+5 | Pipeline Status 탭 | 탭 전환 |
| Alt+6 | Scenario Runner 탭 | 탭 전환 |
| F1 | 컨텍스트 도움말 | 도움말 |
| F11 | 전체 화면 토글 | 화면 |
| Ctrl+/ | 단축키 목록 오버레이 | 도움말 |
