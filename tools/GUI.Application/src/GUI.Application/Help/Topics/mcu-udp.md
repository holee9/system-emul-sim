# MCU/UDP 통신

## UDP 패킷 처리

MCU 계층은 FPGA에서 받은 프레임을 UDP 패킷으로 변환하여 전송합니다.

### 파라미터

| 파라미터 | 범위 | 기본값 | 설명 |
|----------|------|--------|------|
| UDP Port | 1024–65535 | 8000 | 수신 포트 |
| Packet Loss | 0.0–1.0 | 0.0 | 패킷 손실률 |
| Reorder Rate | 0.0–1.0 | 0.0 | 패킷 재정렬률 |
| Corruption Rate | 0.0–1.0 | 0.0 | 패킷 손상률 |

## 시뮬레이션 시나리오

- **정상 통신**: 손실/재정렬/손상 없음
- **손실 시뮬레이션**: PacketLossRate 설정으로 랜덤 패킷 드롭
- **재정렬 시뮬레이션**: 패킷 순서 의도적 변경
