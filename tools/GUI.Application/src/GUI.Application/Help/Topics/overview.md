# 시스템 개요

X-ray Detector Panel System 에뮬레이터는 4계층 파이프라인을 시뮬레이션합니다.

## 아키텍처

```
Panel → FPGA(CSI-2) → MCU(UDP) → Host
```

### 계층 설명

| 계층 | 역할 | 프로토콜 |
|------|------|----------|
| Panel | X선 센서, 물리적 이미지 생성 | 내부 |
| FPGA | CSI-2 데이터 처리, 프레임 버퍼 | CSI-2 |
| MCU | UDP 패킷 전송, 네트워크 처리 | UDP |
| Host | 프레임 수신, 실시간 표시 | SDK |

## 주요 기능

- kVp, mAs, Noise 시뮬레이션
- 패킷 손실/재정렬/손상 시뮬레이션
- 실시간 프레임 프리뷰
- 시나리오 기반 검증
