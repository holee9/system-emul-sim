# Panel 시뮬레이션

## 파라미터 설명

### kVp (킬로볼트 피크)

- **범위**: 40–150 kV
- **기본값**: 80 kV
- **물리적 의미**: X선관의 가속 전압. 높을수록 투과력이 강해집니다.

### mAs (밀리암페어초)

- **범위**: 0.1–500 mAs
- **기본값**: 10.0 mAs
- **물리적 의미**: X선 방사선량. 높을수록 신호 대 잡음비가 향상됩니다.

### NoiseType

- **Gaussian**: 가우시안 분포 노이즈 (Box-Muller 방법)
- **Poisson**: 포아송 통계 노이즈 (X선 광자 계산 시뮬레이션)
- **None**: 노이즈 없음

### DefectRate

- **범위**: 0.0–1.0 (비율)
- **기본값**: 0.001 (0.1%)
- **의미**: 결함 픽셀 비율 (불량 픽셀 시뮬레이션)
