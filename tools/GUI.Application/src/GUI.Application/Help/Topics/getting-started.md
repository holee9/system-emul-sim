# 빠른 시작 가이드

## 10분 퀵스타트

### 1단계: 애플리케이션 시작

애플리케이션 실행 시 자동으로 시뮬레이션 모드로 시작합니다.

### 2단계: 파라미터 설정

**Simulator Control** 탭에서:
- kVp: 80 (기본값)
- mAs: 10.0 (기본값)
- 해상도: 128x128 (기본값)

### 3단계: 파이프라인 시작

**Start** 버튼을 클릭하거나 **Ctrl+R** 을 누릅니다.

### 4단계: 결과 확인

**Frame Preview** 탭에서 실시간 프레임을 확인합니다.
**Status Dashboard** 탭에서 처리 통계를 확인합니다.

## 키보드 단축키

| 단축키 | 기능 |
|--------|------|
| Ctrl+R | 파이프라인 시작 |
| F1 | 도움말 |
| F11 | 전체 화면 |
